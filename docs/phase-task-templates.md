# nimo 研发各阶段任务模板

> 对标华为IPD/OPPO消费电子研发体系，结合nimo智能眼镜产品特点。
> 任务按职能模块划分，含依赖关系、工期估算、交付物、门控必要性。

---

## 阶段概览

| 阶段 | 任务数 | 典型周期 | 核心目标 |
|------|--------|----------|----------|
| Concept/立项 | 15 | 2-4周 | 明确需求、评估可行性、立项审批 |
| EVT | 45 | 10-16周 | 工程验证、功能样机、打通技术路径 |
| DVT | 38 | 8-14周 | 设计验证、可靠性测试、开模 |
| PVT | 30 | 6-10周 | 量产验证、试产、认证 |
| MP | 12 | 4-6周 | 量产放行、持续监控 |

## 六、任务依赖关系总图

### 关键路径（平台项目典型）

```
G0立项评审
  ├→ 硬件设计线: 系统架构→原理图→评审→Layout→评审→打板→贴片
  ├→ 光学设计线: 波导设计→仿真→光学样品
  ├→ 结构设计线: ID设计→镜腿结构→堆叠设计→评审→手板
  └→ 软件设计线: 固件架构→BSP/驱动→蓝牙协议

  以上并行，汇聚于→ EVT样机组装 → 全面测试 → G1评审

G1→ 设计优化(并行) → 开模 → DVT样机 → 可靠性测试+认证预测试 → G2评审
G2→ 产线准备+试产 → 良率验证+正式认证 → G3评审
G3→ 量产BOM冻结 → 首批量产 → OQC/ORT → G4量产放行
```

### 平台项目 vs 产品项目任务差异

**平台项目包含全部任务（上述所有）**

**产品项目仅需：**
- 镜架结构设计与开模
- 桩头适配验证
- 产品特有件BOM编制（继承平台BOM + 补充差异件）
- 外观/颜色/材质验证
- 整机组装验证
- 产品级认证（如有差异）
- 各阶段门控评审
