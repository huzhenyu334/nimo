## Part 1: 战略规划（SP）

### 1.1 执行摘要（一页纸）

**公司现状**：BitFantasy（品牌nimo）成立于2025年5月，融资3,600万元，20人研发型团队，首款AI智能眼镜2025年3月上市。当前零营收、零客户、零渠道。

**核心判断**：中国AI音频眼镜市场存在12-15个月窗口期（Meta未入华、华为AI弱、小米/OPPO尚未入场）。这是nimo从0到1的唯一机会。

**战略定位**：**「中国AI体验最好的智能眼镜」**——不做AR显示，不做B2B，不做海外（首年），集中全部资源做好中文AI体验+极致性价比+线上DTC。

**关键数字**：
| 指标 | 目标 |
|------|------|
| 首年营收 | 900-1,440万元（5,000-8,000副 × ASP 1,800元） |
| 毛利率 | 55-65% |
| 定价区间 | 1,499-1,999元 |
| 年度总预算 | 800万元（占融资22%） |
| 月烧上限 | 200万元 |
| 跑道 | 18个月（至2026年11月） |
| 年末剩余资金 | ≥1,200万元 |

**三大战略方向**：
1. **AI体验碾压**（投入300万，37%）—— 与智谱深度合作，做中文AI眼镜体验第一
2. **极致性价比+DTC**（投入250万，31%）—— 比华为便宜30%，纯线上省渠道成本
3. **场景化AI功能**（投入150万，19%）—— 翻译先行，OTA扩展，复用硬件平台

**生死线**：3月产品必须准时上市。每延迟1个月 = 损失200万现金 + 窗口期缩短。

**止损线**：上市后3个月月销＜100副，立即转型。

### 1.3 五看分析

#### 看趋势：必须跟 vs 可忽略

**必须跟：**
- **AI眼镜化**（投入300万）—— Meta Ray-Ban年出货1,000万+副验证品类成立，品类从「显示设备」转向「AI随身助手」不可逆
- **中文AI多模态能力**（投入200万）—— 国内大模型（智谱GLM-4V、月之暗面Kimi）已接近GPT-4V水平，打造「中文AI体验最好的智能眼镜」

**可忽略：**
- AR光波导/空间计算 —— Xreal已投入3亿美元+600人，20人团队做不了
- 自研芯片 —— 高通公版方案已够用
- 企业级AR头显 —— 与团队能力严重不匹配

#### 看客户：目标客群与需求取舍

**All-in客群：25-35岁中国城市科技消费者（男性为主）**
- 一二线城市，月收入1.5万+，B站/小红书/即刻活跃用户
- 潜在兴趣人群300-500万人，消费力匹配1,499-1,999元
- 所有营销资源集中在B站测评+小红书种草+即刻口碑

**产品底线**：外观像普通眼镜（<50g）、音频体验好、AI助手好用（响应<2秒）、续航≥6小时、价格1,499-2,499元

**明确放弃**：AR显示、处方镜片（首代）、运动防水、儿童/老年、游戏沉浸

#### 看竞争：在哪打 vs 避开哪里

**集中兵力的维度：**
1. **中文AI体验** —— 语音识别>95%、多模态响应<3秒、连续对话
2. **性价比** —— 定价1,499-1,999元，BOM 500-700元，毛利率55-65%
3. **迭代速度** —— 上市后45天首次OTA，90天第二次OTA

**坚决避开：**
- 不和Meta比品牌、不和Xreal/Rokid比AR显示、不和华为比渠道
- 不打价格战到千元以下、不做B2B

**差异化定位：「中国最好用的AI眼镜」—— AI体验最好、价格合理的智能眼镜。**

#### 看机会：最值得押注的3个窗口

| 优先级 | 机会 | 时间窗口 | 投入 | 预期回报 |
|--------|------|----------|------|----------|
| 1（All-in） | 中国AI音频眼镜空白期 | 2025.3-2026.6 | 500万 | 首年营收900-1,440万 |
| 2（功能模块） | 实时翻译眼镜 | 2025.Q3-2026.Q4 | 150万 | 增量2,000-3,000副/年 |
| 3（小赌注） | 设计师品牌联名 | 2025.Q4-2026.Q2 | 100-150万 | 营收125-300万+品牌破圈 |

**明确不押注**：海外DTC（首年不做）、企业级AR、头部时尚品牌联名

#### 看自己：能力边界

**圈内（可依赖）**：嵌入式硬件开发、工业设计（3人团队）、APP开发、小团队快速迭代

**圈外（需外部补充）**：
- AI大模型 → 合作智谱（200万），不自建
- 品牌营销 → 招1人+外包（100万/年）
- 电商运营 → 招1人+开店启动（50万）
- 供应链 → 加招1-2人（50-70万/年）
- 售后 → 外包客服（10-15万/年）

**铁律**：同时只能做1件事做好，不自建AI团队，不开线下店，不招超过5人。

### 1.4 战略意图与目标

#### 愿景
成为中国AI智能眼镜品类的定义者——让每个人都拥有一副懂中文、会思考的AI眼镜伙伴。到2030年，nimo是中国消费者提到「AI眼镜」时第一个想到的品牌，年出货量突破100万副。

#### 一句话战略
**做「中国AI体验最好的智能眼镜」，1,499-1,999元价位，首年聚焦国内线上市场，单品打爆，活过18个月。**

#### 分年目标

| 维度 | 2025年（生存年） | 2026年（增长年） | 2027年（突破年） |
|------|------------------|------------------|------------------|
| 营收 | 900-1,400万 | 4,000-6,000万 | 1.5-2.5亿 |
| 出货量 | 5,000-8,000副 | 2-3万副 | 8-12万副 |
| 市占率 | AI音频眼镜前3 | 品类第1（>30%） | 中国AI眼镜>20% |
| 产品 | 1款+2次OTA | 二代+联名款 | 2-3个SKU |
| 团队 | 20→25人 | 35-40人 | 50-60人 |
| 融资 | Q4启动A轮接触 | 完成A轮5,000万-1亿 | 实现季度盈利 |

#### 三大战略方向

**方向一：AI体验碾压（投入300万，占37%）**
- 2周内签约智谱为首选AI合作伙伴
- 端侧优化：语音识别>95%、多模态响应<3秒、连续对话
- 上市后45天首次OTA（AI增强），90天第二次OTA（翻译模式）
- 建立用户反馈→AI优化数据飞轮，每2周迭代prompt工程

**方向二：极致性价比+DTC（投入250万，占31%）**
- 定价1,499-1,999元，BOM 500-700元，毛利率55-65%
- 纯线上DTC：官网+天猫+京东+拼多多
- 供应链优化100万，目标BOM降10-15%
- 众筹预热，目标上市首月500副

**方向三：场景化AI功能矩阵（投入150万，占19%）**
- Q2：翻译模式OTA上线（中英/中日/中韩）
- Q3：会议记录模式上线（实时转写+摘要）
- Q4：开放第三方AI应用接入
- 与携程/飞猪联合营销（翻译眼镜×出境游）

#### 明确放弃清单

| 类别 | 不做的事 | 原因 |
|------|----------|------|
| 产品 | AR显示眼镜 | Xreal 600人+3亿美元在做，20人团队入场是送死 |
| 产品 | 第二款硬件（首款验证前） | 分散资源 |
| 产品 | 自研芯片 | 高通公版够用 |
| 市场 | 海外市场（2025年） | 需200-400万+本地化 |
| 市场 | B2B/企业级 | 需解决方案团队+长销售周期 |
| 市场 | 线下渠道 | 纯线上DTC |
| 市场 | 价格战到千元以下 | 毛利不支持 |
| 团队 | 自建AI团队 | 需10人+2,000万/年，烧钱黑洞 |
| 团队 | 招超过5人 | 月烧控制在200万以内 |

### 1.5 业务设计

#### 客户选择
- **第一优先**：25-35岁城市科技男性消费者（潜在兴趣人群300-500万人）

### 2.4 风险与预案

#### Top 3 风险

**风险1：产品质量/体验翻车（概率25%，影响：致命）**
- **触发条件**：上市后60天内退货率>8%或差评率>15%
- **预防**：200副内测机种子用户测试；品控专员驻厂，关键工序100%检测
- **应对**：立即暂停发货+召回+全检（成本10-20万）；设计缺陷则暂停销售，2周内出修复方案
- **止损线**：首月退货率>15%，立即暂停发货排查

**风险2：大厂提前入场碾压（概率20%，影响：高）**
- **触发条件**：小米/OPPO以999元价格+品牌号召力入场
- **预防**：抢先建立「中文AI体验最好」心智
- **应对A**：转向场景深耕（翻译眼镜/会议眼镜），做垂直品类第一
- **应对B**：考虑与大厂谈AI方案授权/ODM合作

**风险3：现金流断裂（概率15%，影响：致命）**
- **触发条件**：月burn rate连续3个月>300万或Q3营收仍<200万
- **预防**：每月财务review，超支立即裁项目
- **安全线**：年末剩余≥1,200万（6个月跑道），低于此线立即启动融资
- **融资Plan B**：①政府产业基金 ②智谱/高通生态基金 ③供应链金融
- **极端方案**：剩余800万时启动「卖身」谈判，不等到0

#### 供应链风险
- **触发条件**：核心物料交期延误>2周
- **预防**：Q2前每个核心物料至少2家合格供应商；保持4-6周安全库存
- **应对**：启用备选供应商，接受成本上浮10-15%保交期
